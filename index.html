História da WEB

"Uma das maiores criações da humanidade, a World Wide Web abre possibilidades ilimitadas para a disseminação do conhecimento e a comunicação global."

Nelson Mandela

A World Wide Web, ou simplesmente a WEB, é uma rede global de documentos interconectados, criada por Tim Berners-Lee em 1989. Ela revolucionou a maneira como as pessoas
compartilham informações e se comunicam. A Web permite que os usuários acessem e compartilhem conteúdo por meio de navegadores, usando hiperlinks. Desde então, ela cresceu
exponencialmente, incluindo a evolução de mídias sociais, comércio eletrônico e serviços online, moldando profundamente a sociedade e a economia. Hoje, a Web continua
a se expandir, impulsionando a inovação e a conectividade global.

O que são Clients

No contexto de "clients x server" (clientes x servidor), "clients" referem-se aos dispositivos ou sistemas que solicitam serviços ou recursos a um servidor em uma rede de
computadores. Os clientes são responsáveis por enviar pedidos ou requisições ao servidor, buscando acesso a informações, serviços, ou funcionalidades hospedados nele.

Eles podem ser computadores pessoais, smartphones, tablets, ou qualquer dispositivo conectado à rede que interaja com o servidor. Os clientes geralmente utilizam
aplicativos ou programas específicos para enviar solicitações, como navegadores da web, aplicativos de email, ou aplicativos de redes sociais. A relação entre clientes e
servidor é fundamental na arquitetura cliente-servidor, onde o servidor fornece os recursos necessários para atender às solicitações dos clientes, tornando possível a
troca de informações e serviços na internet e em redes locais.

O que são Servers

No contexto de "clients x server" (clientes x servidor), "servers" (servidores) são sistemas ou dispositivos de computação dedicados que fornecem recursos, serviços ou
informações para os clientes em uma rede. Eles desempenham um papel central na arquitetura cliente-servidor, atendendo às solicitações dos clientes. Os servidores podem
assumir diversas formas, como servidores web, servidores de email, servidores de banco de dados, e muitos outros, cada um com sua função específica.

Os servidores são projetados para serem confiáveis, seguros e capazes de lidar com múltiplas solicitações simultaneamente. Eles armazenam e gerenciam dados, aplicativos
ou serviços que os clientes precisam acessar, garantindo que as informações sejam entregues eficientemente e com disponibilidade constante. Em resumo, os servidores são
a infraestrutura central que permite a comunicação e o compartilhamento de recursos entre os clientes em uma rede.

Linguagens de Programação

Linguagens de programação são meios de comunicação entre humanos e computadores, permitindo a criação de software por meio de instruções compreensíveis tanto para
programadores quanto para máquinas. Cada linguagem tem sua sintaxe e semântica únicas. Alguns exemplos:

  1. Python: Conhecida pela legibilidade, é amplamente usada em desenvolvimento web, análise de dados e aprendizado de máquina.
  2. JavaScript: A principal linguagem para desenvolvimento web, permite criar interatividade em sites e aplicativos.
  3. Java: Usada em aplicativos empresariais, dispositivos móveis Android e sistemas embarcados.
  4. C++: Utilizada para desenvolver jogos, sistemas operacionais e aplicativos de alto desempenho.
  5. Ruby: Valorizada pela simplicidade e produtividade, é usada em desenvolvimento web, com o framework Ruby on Rails.
  6. Swift: Criada pela Apple, é usada para desenvolver aplicativos iOS e macOS.
  7. C#: Amplamente usada na criação de jogos com o motor Unity e em desenvolvimento de aplicativos Windows.

Cada linguagem tem suas vantagens e é escolhida de acordo com o contexto e os requisitos do projeto.

Ferramentas Utilizadas

As principais ferramentas utilizadas na programação web incluem:

  1. Editores de Código: Como VS Code, Sublime Text e Atom, que permitem escrever e editar código de maneira eficiente.
  2. Navegadores: Como Chrome, Firefox e Safari, que são usados para testar e depurar aplicativos Web.
  3. Frameworks: Como React, Angular e Vue.js, que simplificam o desenvolvimento de interfaces de usuário dinâmicas.
  4. Servidores Web: Como Apache e Nginx, que hospedam sites e aplicativos web.
  5. Bancos de Dados: Como MySQL, PostgreSQL e MongoDB, que armazenam e gerenciam dados.
  6. Controle de Versão: Como Git e GitHub, que rastreiam alterações no código-fonte.
  7. Ferramentas de Design: Como Adobe XD e Sketch, para criar layouts e protótipos de design.
  8. Ferramentas de Teste: Como Selenium e Jest, para testar a funcionalidade do aplicativo.

Essas ferramentas são cruciais para o desenvolvimento web moderno, tornando-o mais eficiente e produtivo.

Inspetor de Elementos

O "Inspetor de Elementos" é uma ferramenta poderosa disponível em navegadores da web, como Google Chrome, Firefox e outros. Ele permite aos desenvolvedores e usuários
visualizar e interagir com o código HTML, CSS e JavaScript de uma página da web em tempo real. Você pode acessá-lo clicando com o botão direito do mouse em qualquer parte
de uma página da web e selecionando "Inspecionar" ou pressionando a tecla F12.

O Inspetor de Elementos permite examinar a estrutura da página, editar e depurar código, visualizar recursos carregados, analisar estilos CSS, monitorar solicitações de rede
e muito mais. É uma ferramenta essencial para desenvolvedores da web, pois ajuda a diagnosticar problemas, testar alterações e entender como as páginas da web são construídas.
Além disso, é útil para usuários comuns que desejam entender como um site funciona ou fazer pequenas modificações temporárias na aparência de uma página da web.

Estrutura Básica do HTML

A estrutura básica do HTML (Hypertext Markup Language) consiste em elementos que compõem uma página da web. Inicia com o elemento < html >, que engloba todo o conteúdo da página.
Em seguida, temos o < head >, onde informações sobre a página, como o título, metadados e links para estilos, são definidas. O < body > contém o conteúdo visível da página, como texto,
imagens, links e outros elementos.

Os elementos de marcação, como < p > para parágrafos ou < h1 > para cabeçalhos, são usados dentro do < body > para estruturar o conteúdo. Tags têm abertura < tag > e fechamento < /tag >,
com exceção de algumas, como < img >.

Principais Tags

As principais tags HTML são fundamentais para estruturar e formatar conteúdo em páginas da Web. Algumas delas incluem:

  • < html >: Define o início e o fim de um documento HTML.
  • < head >: Contém informações sobre a página, como título, metadados e links para estilos e scripts.
  • < title >: Especifica o título da página exibido na barra de título do navegador.
  • < meta >: Fornece metadados, como conjunto de caracteres e descrições para mecanismos de busca.
  • < body >: Contém o conteúdo visível da página, como texto, imagens e elementos interativos.
  • < h1 >, < h2 >, < p >: Tags para cabeçalhos, parágrafos e outros elementos de texto.
  • < a >: Cria links para outras páginas ou recursos.
  • < img >: Exibe imagens na página.
  • < ul >, < ol >, < li >: Usadas para criar listas não ordenadas e ordenadas.
  • < div > e < span >: Elementos de divisão usados para agrupar e estilizar conteúdo.

Essas tags são essenciais para criar estrutura e semântica em páginas da web, permitindo que os navegadores interpretem e exibam o conteúdo de forma adequada.

Atributos das Tags

Os atributos são informações adicionais que podem ser fornecidas às tags HTML para controlar seu comportamento ou fornecer mais detalhes. Eles são definidos dentro das tags com
um nome e um valor. Por exemplo, em < a href="https://www.exemplo.com" >, "href" é um atributo que define o link para "https://www.exemplo.com". Os atributos podem
modificar a aparência, comportamento ou funcionalidade de elementos HTML, como definir cores em elementos de estilo, especificar links ou fornecer informações adicionais para
acessibilidade. Eles desempenham um papel crucial na personalização e no funcionamento correto das páginas da web.

Textos

Em HTML, os textos são representados principalmente por meio da tag < p > para parágrafos e tags de cabeçalho, como < h1 >, < h2 >, etc., para títulos. O conteúdo de texto é colocado
entre as tags de abertura e fechamento, e pode incluir formatação básica, como negrito < strong >, itálico < em >, listas < ul > e < ol >, além de links < a > para outras páginas. Também é
possível utilizar estilos CSS para formatar o texto, como cores e tamanhos de fonte. O HTML é usado para estruturar e apresentar informações de texto em páginas da web, tornando-o
legível e acessível.

Listas Ordenadas e Não Ordenadas

Em HTML, listas ordenadas < ol > são usadas para criar listas numeradas, onde cada item é precedido por um número sequencial. Já as listas não ordenadas < ul > criam listas de itens
com marcadores, como pontos ou símbolos, em vez de números. Cada item da lista é definido com a tag < li >. Listas ordenadas são úteis quando a ordem é relevante, como em instruções
passo a passo, enquanto listas não ordenadas são adequadas para itens sem uma sequência específica, como uma lista de recursos. Ambas são elementos HTML que ajudam a organizar e estruturar
informações em páginas da web.

(Voltar ao Início)

Links

Em HTML, os links são criados com a tag < a >, que permite aos usuários navegar para outras páginas ou recursos na web. O atributo "href" dentro da tag define o destino do link, seja
uma URL externa, um arquivo local ou uma âncora em uma página. Links são vitais para a navegação e conectividade na web, permitindo que os usuários acessem diferentes conteúdos e
recursos com facilidade. Eles são uma parte fundamental do hipertexto, que é a base da World Wide Web, tornando possível a interconexão de informações em todo o ambiente online.
